# Projeto Financeiro

Este projeto é uma aplicação web para gerenciar transações financeiras.

## Estrutura do Projeto

- `banco/`: Scripts de criação do banco de dados.
- `backend/`: Código Laravel.
- `frontend/`: Código Angular.
